# The Sourdough Field Guide

This version uses separate pages for each main section and separate pages for each recipe.

## Main pages

- `index.html` home
- `calculator.html` starter calculator
- `basics.html` starter basics and ratio guide
- `schedule.html` loaf schedule
- `recipes.html` field plate recipe catalog
- `troubleshooting.html` troubleshooting

## Recipe pages

Individual recipes live in the `recipes/` folder.

Current recipe pages:

To edit a recipe, open that recipe's HTML file in GitHub, edit the text, and commit changes. Netlify redeploys automatically.

## Recipe illustrations

Field-guide illustrations live in `assets/illustrations/`.

The recipe catalog uses the field plates as recipe cards. Individual recipe pages stay cleaner and use smaller thumbnail plates only where art already exists.


## Submit recipe form

`submit.html` contains a Netlify form named `recipe-submission`.
Submissions can be reviewed in Netlify under the site's Forms area.
The form does not automatically create recipe pages; it collects ideas for review.

## Recipe pages

Each recipe has its own file in the `recipes/` folder.
The long-form recipe is styled as the field guide entry.
The quick recipe button switches to a plain kitchen-use version.
